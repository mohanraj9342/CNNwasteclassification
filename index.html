About This Project

An end-to-end deep learning solution for automated waste classification

Problem Statement

Proper waste segregation is crucial for environmental sustainability and composting. Manual sorting is time-consuming and error-prone. This project demonstrates how AI can automate the biodegradability classification process.

Solution

I developed a Convolutional Neural Network (CNN) that can accurately classify waste images into two categories:

  • Biodegradable: Organic materials that naturally decompose (food waste, leaf waste, paper waste, wood waste)
  • Non-Biodegradable: Materials that don't naturally decompose (plastic bottles, metal cans, e-waste, plastic bags)

Methodology

Detailed approach to building the waste classification system

1

Data Collection & Preprocessing

Downloaded dataset using kagglehub, organized nested subcategories into binary classification, and applied image preprocessing including resizing to 224x224 pixels and normalization.

path = kagglehub.dataset_download("aashidutt3/waste-segregation-image-dataset")
2

Data Augmentation

Applied various augmentation techniques to increase dataset diversity and prevent overfitting: rotation, width/height shifts, shearing, zooming, and horizontal flipping.

  • Rotation: ±20 degrees
  • Shifts: 20% width/height
  • Zoom: 20% range
  • Horizontal flip enabled
3

Model Architecture

Designed a CNN with four convolutional blocks, each containing Conv2D, BatchNormalization, and MaxPooling layers, followed by fully connected layers with dropout for regularization.

Conv2D(32) → BatchNorm → MaxPool
Conv2D(64) → BatchNorm → MaxPool
Conv2D(128) → BatchNorm → MaxPool
Conv2D(256) → BatchNorm → MaxPool
Flatten → Dense(512) → Dense(1) + Sigmoid
4

Training & Optimization

Used Adam optimizer with learning rate scheduling, early stopping, and cross-validation for optimal performance. Trained for 30 epochs with early stopping achieving 95.38% validation accuracy.

Adam Optimizer Binary Crossentropy Loss Early Stopping Learning Rate Scheduling
