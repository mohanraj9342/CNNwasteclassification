# Waste Classification using CNN

This repository contains a Jupyter Notebook for classifying waste images as **Organic** or **Recyclable** using a Convolutional Neural Network (CNN) built with TensorFlow/Keras.

## Repository Structure

```
CNNwasteclassification/
│
├── Googlecolab_wasteclassification.ipynb   # Main notebook with all code
├── requirements.txt                        # List of required Python packages
├── README.md                               # Project documentation
└── .gitignore                              # Git ignore file
└──  To develop a CNN model to classify images of plastic waste.pptx   #PPT file for better understand
```

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/mohanraj9342/CNNwasteclassification.git
   cd CNNwasteclassification
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the notebook**
   - Open `Googlecolab_wasteclassification.ipynb` in Jupyter Notebook, JupyterLab, or VS Code.
   - Follow the instructions in the notebook to download the dataset and train the model.

## Dataset

- The dataset is automatically downloaded from Kaggle using `kagglehub` within the notebook.
- Source: [Kaggle - Waste Classification Data](https://www.kaggle.com/datasets/techsash/waste-classification-data)
